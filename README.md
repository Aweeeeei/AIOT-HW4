# 🏦 Massive 美股即時新聞中譯摘要助手

這是一個基於 Streamlit 的 AI 金融新聞應用程式。

本系統利用 **Massive REST API** 抓取指定美股代號（如 TSM, NVDA, AAPL）的財經新聞，透過 **LSA 演算法** 進行重點萃取，並結合 **Google Translate** 自動將摘要翻譯為繁體中文。

## 📄 摘要

第 9 週「檢索增強生成 (RAG)」與 第 12 週「打造 AI Web APP」之延伸實作

### 題目：透過 Massive API 實作美股即時新聞摘要

實作延伸自課程**第 9 週「檢索增強生成 (RAG)」**關於外部資料檢索的概念，以及**第 12 週「打造 AI Web APP」**的部署實務。在生成式 AI 的應用場景中，如何讓 AI「即時獲取」並「精準處理」外部資訊是關鍵挑戰。鑑於美股資訊多為英文且更新頻繁，一般投資人常面臨語言隔閡與資訊過載的痛點，本專案開發了一套針對特定美股代號（Ticker）的自動化新聞速覽系統。

有別於傳統 LLM 生成的高延遲，本系統以「速度」為首要考量。技術架構上，採用 **Massive REST API** ，確保能精準鎖定如 TSM、NVDA 等個股資訊。在後端處理，本實作導入**多執行緒 (Multi-threading)** 平行運算技術，同時執行資料下載、LSA 萃取式摘要與自動化中英翻譯。

實測結果顯示，系統能在 **數秒內** 完成多篇長篇財經新聞的檢索與中譯摘要，成功在 Streamlit 雲端環境下實現了低延遲、高效率的跨語言資訊服務，體現了將 AI 技術轉化為實用 Web 應用的工程價值。
